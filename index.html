Endpoints and their PARAMETERS

HTTP Method Endpoint Parameter(s) Description
get /ontologies/get - Get the list of all ontologies hosted.
Post /ontology/create json_data Construct an ontology from input data
post /node/insert json_data Insert a new node in the ontology
get /nodes - Get the list of all the nodes present in the ontology
get /node/id node_id Get a specific nodes present in the ontology
/delete /node/delete id Delete a specific ontology
get /node/search sentence_to_search Find a node with given characteristics.
post /node/update json_data Update a node with new information.

Errors

The CAOGen API uses the following error codes:

Error Code Meaning
X000 Some parameters are missing. This error appears when you don't pass every mandatory parameters.
X001 Unknown or invalid resource. This error appears when the requested resource is not found.
X002 invalid endpoint. This error appears when the uri is not correct or is invalid.
X003 Server error . This error appears if the server is not responding.
